# ms

[ms](https://github.com/vercel/ms) 라이브러리를 한국어로 사용하기위해 수정하였습니다.

## 설치

```sh
npm i @wonderlandpark/ms
```

## 예시

```js
ms('2일')  // 172800000
ms('1d')      // 86400000
ms('10h')     // 36000000
ms('2.5시간') // 9000000
ms('2h')      // 7200000
ms('1분')      // 60000
ms('5초')      // 5000
ms('1년')      // 31557600000
ms('100')     // 100
ms('-3일') // -259200000
ms('-1시간')     // -3600000
ms('-200')    // -200
```

### 밀리초에서 변환

```js
ms(60000)             // "1m"
ms(2 * 60000)         // "2m"
ms(-3 * 60000)        // "-3m"
ms(ms('10시간'))    // "10h"
```

### 시간 형식 변환

```js
ms(60000, { long: true })             // "1분"
ms(2 * 60000, { long: true })         // "2분"
ms(-3 * 60000, { long: true })        // "-3분"
ms(ms('10 시간'), { long: true })    // "10시간"
```
